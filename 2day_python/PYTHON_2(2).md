### 📂 2. 반복문 

> while 종료 조건에 해당하는 코드를 직접 작성하여 반복문을 종료시켜야함 

> 조건이 참인 경우 들여쓰기 되어있는 코드 블록이 실행됨

> 코드 블록이 모두 실행되고 다시 조건식을 검사하며 반복적으로 실행됨

> while문은 무한 무한 루프를 하지 않도록 종료조건이 반드시 필요 

```python
a = 0

while a < 5:	
	print(a)
	a += 1
print('끝')
```
